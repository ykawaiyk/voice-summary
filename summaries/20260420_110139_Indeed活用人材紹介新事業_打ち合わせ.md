## 要点

### 事業モデルの概要
Indeedを人材紹介のフロント集客ツールとして活用する代理店モデル。

**フロー：**
1. 企業から「Indeedへの広告掲載許可」を取得
2. 弊社が広告費を負担してIndeedに求人掲載（給与条件を高めに設定してCPAを下げる）
3. 応募者が面談に来る
4. その企業にマッチすればそのまま紹介、マッチしなければ別の企業に紹介して紹介料を取る
5. 繰り返しで紹介料が積み上がる

**収益構造：**
| 項目 | 数値 |
|------|------|
| CPA（着座1件） | 約1万円 |
| 成約率（面談→紹介決定） | 約30% |
| 紹介手数料 | 35% |
| 初期テスト予算 | 30万円（30流入見込み） |
| コンサバ成約件数 | 2件 |
| 1件あたりの売上 | 約80万円 |
| テスト期間の想定売上 | 約160万円（2件×80万） |

### 主力案件・掲載候補企業
- **渋谷ロクシタンカフェ**：渋谷のど真ん中で集客力が抜群。Indeed経由の応募が大量に見込める。カフェ応募者を牛丼チェーンや他業態に横展開する動きも想定
- デフワークス焼肉（全国展開）
- クシカツタナタタ
- ゴーリングス（複数ブランド展開）：クックビズと同様の仕組みで攻める
- カッポーズ（牛丼）：契約済み、120万円クラスの高単価案件

### Indeed活用の注意点
- Indeedは求人掲載の締め出し（利用規制）が強化されており、企業が自社採用を優先するよう誘導している
- **対策**：企業から「Indeedへの掲載許可と広告費の代理払い」を明示的に合意取得する必要あり
- **候補者クレーム対策**：「その求人は見送りになったため別を紹介」という形のスクリプトで自然に横展開できる流れを整備する

### プラットフォーム選定
- **ジョビンズ**（成果報酬型・10%/10%分配）を採用
  - 月額費用がかかるサーカス等は不採用（固定費を嫌う方針）
  - ジョビンズ上でHAMIDERUアカウントにログインして候補者を提案する流れ
- 候補者がサーカス等でも決まらない場合の受け皿としても活用
